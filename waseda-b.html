わせだ塾のこだわり

わせだ塾 塾長

早稲田大学教育学部卒 / 翻訳家 / 25年の指導実績

「教育とはものごとの暗記ではなくて自分の心で考える練習なんです」(アインシュタイン)

富山県に生まれて最も幸運だったと思うことは代々木ゼミナール校長の木下先生に出会えたことです。いつでも生徒を笑わせようとしていた姿を今でも覚えています。先生は英語や現代文を教える天才でした。私も先生のレベルに追い付こうと日々邁進しています。

大学ではまたしても幸運なことに新田章という偉大な哲学者に出会え、公私を超えて指導して頂き学問とは何かを教えて頂きました。

ロンドンに留学し外国人に日本語を教えながら英会話を学び翻訳や通訳の仕事もできるようにもなりました。現代は地球のどこであれ言語と民族が交わる時代です。どの科目も歴史的な視点で指導していきたいです。

5教科を自分の力がおよぶ範囲で教えています。不登校や障害を持つ子も入れますが少しだけ面接があります。どんな生徒も見捨てません。

COLUMN
COLUMN

塾長のコラム

私が研究する分野のコラムを3つ書きました。学校ではまだ教えていない範囲もあるので知らない単語があったら検索してください。

日本地図の形は世界地図に似ているということを知っていますか？北海道はユーラシア大陸、本州はアフリカ大陸、四国はオーストラリア大陸、九州は南北アメリカ大陸の形に似ているのだそうです。

この法則はフラクタルといいます。全体の中に部分があって、部分の中にも全体がある。たとえば木の枝は木全体の形に似ていたり、海岸線のギザギザは拡大しても同じギザギザの形をしています。ブロッコリーの房も同じ形の繰り返しです。

フラクタルは自然界のいたるところに見られます。数学者のマンデルブロがこの概念を提唱しました。物理学でもフラクタルの考え方は重要で、カオス理論やブラウン運動の研究に応用されています。

地理を学ぶことは物理を学ぶことにもつながります。地球の形、気象のパターン、地殻変動、すべてが物理法則に従っています。教科の壁を超えて学ぶことの大切さがここにあります。

円周率(π)は3.14159265...と永遠に続く無限小数ですが、面白い公式があります。

π/4 = 1 - 1/3 + 1/5 - 1/7 + 1/9 - ... (ライプニッツの公式)

分数の足し算と引き算を永遠に繰り返すだけで円周率が出てくるのです。これは算数と幾何学が深いところでつながっていることを示しています。

さらにオイラーの公式 e^(iπ) + 1 = 0 は「数学で最も美しい公式」と呼ばれています。ネイピア数e、虚数i、円周率π、1と0という数学の最も重要な5つの定数がひとつの式に集約されているからです。

リーマン予想は素数の分布に関する未解決問題で、解いた人には100万ドルの賞金が出ます。素数は2, 3, 5, 7, 11, 13...と不規則に現れますが、リーマンのゼータ関数を使うと素数の出現パターンが見えてくるのだそうです。

算数の基礎がしっかりしていれば、こうした高度な数学にもつながっていきます。理科の実験データを分析するにも数学は不可欠です。算数と理科は車の両輪のような関係なのです。

第二次世界大戦後、日本は平和憲法を制定し戦争を放棄しました。しかし世界ではその後も多くの紛争が起きています。朝鮮戦争、ベトナム戦争、湾岸戦争、そして現在も続く地域紛争。

冷戦時代はアメリカとソ連が世界を二分していました。1991年にソ連が崩壊し冷戦は終結しましたが、新たな対立構造が生まれています。

日本国憲法の三大原則は国民主権、基本的人権の尊重、平和主義です。これらは戦争の反省から生まれたものですが、時代とともにその解釈も変化しています。

公民を学ぶということは、自分が社会の一員としてどう生きるかを考えることです。選挙権、納税の義務、表現の自由、すべてが私たちの日常生活に直結しています。

歴史は暗記科目だと思われがちですが、本当は「なぜそうなったのか」を考える科目です。過去の出来事の因果関係を理解することで、現在の社会問題を多角的に見る力が身につきます。

グローバル化が進む現代において、日本の歴史だけでなく世界の歴史を知ることはますます重要になっています。異なる文化や価値観を理解するための第一歩が歴史の学習なのです。
